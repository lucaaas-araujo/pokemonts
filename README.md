# Projeto PokemonTS

Projeto desenvolvido em **TypeScript** com o objetivo de praticar conceitos de:

- Classes
- Herança (subclasses)
- Classes abstratas
- Organização de código orientado a objetos

---

## 🚀 Tecnologias utilizadas

- TypeScript
- Node.js

---

## 📦 Pré-requisitos

Antes de começar, você vai precisar ter instalado:

- Node.js (versão 16 ou superior recomendada)
- npm (geralmente já vem com o Node)

---

## 📥 Instalação

Clone o repositório:

```bash
git clone https://github.com/lucaaas-araujo/pokemonts.git
```

Entre na pasta do projeto:

```bash
cd pokemonts
```

Instale as dependências:

```bash
npm install
```

## ▶️ Rodando o projeto
```bash
ts-node Menu.ts
```
